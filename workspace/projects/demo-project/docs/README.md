# Demo Project

This is a sample project under `workspace/projects/demo-project/`.

## Structure

- `src/` — Generated source code
- `docs/` — Specifications and notes
- `logs/` — Agent run traces
- `metadata.json` — Execution status and timestamps

## Next Steps

1. Open the Cockpit dashboard (`cd cockpit && npm run dev`)
2. Select files in the Project Explorer
3. Submit agent prompts from the center pane
4. Monitor live telemetry in the Agent Monitor feed
